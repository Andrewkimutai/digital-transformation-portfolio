  # Business Request Management System
  
  ## Overview
  
  A Python-based business workflow application that allows employees to submit requests and managers to review and approve them.
  
  ## Features
  
  - Create requests
  - View requests
  - Approve requests
  - Reject requests
  - Persistent storage using JSON
  
  ## Technologies
  
  - Python
  - JSON
  
  ## Business Value
  
  This project demonstrates how simple automation can improve organizational processes by tracking and managing requests efficiently.
